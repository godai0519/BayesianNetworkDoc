Detailed Description

template<class Elem>
class bn::matrix< Elem >

Provide a class representing dinamic N dimensions (N is equal to or greater than 1).

Dimensions can be determined at run time. Element access can be performed by std::vector<std::size_t> instead of operetors [][]...[].

Template Parameters
ElemType of elements hold by matrix<Elem>.
