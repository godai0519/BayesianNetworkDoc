◆ topological_sort()

template<class RepresentMethod >
std::vector<component::node_ptr> bn::topological_sort ( network< RepresentMethod > const &  g)

Perform topological sort for given graph (however, a reversed series can be obtain).

The topological sort gets a node series such that if a edge (u,v) is exist, v appears after u. This function is create a `‘reversed’' topological sort series. The time complexity is O(V + E).

Template Parameters
RepresentMethodbn::adjacency_list or adjacency_matrix.
Parameters
[in]ga network including nodes which are targets of sort.
Returns
A node series created by `‘reversed’' topological sort.
